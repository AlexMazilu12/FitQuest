# FitQuest - Backend

A fitness tracking and management REST API built with Java Spring Boot, developed as an individual project for the 3rd semester at Fontys University of Applied Sciences.

## Features

- **Authentication & Authorization:** JWT-based user authentication with role-based access control (user, trainer, admin)
- **Workout Management:** Create, customize and manage workout plans
- **Client Management:** Trainers can manage and track their clients
- **REST API:** Full RESTful API consumed by the React frontend
- **Testing:** Full coverage including unit, integration and E2E testing with Cypress
- **CI/CD:** Automated GitLab pipelines for continuous integration and deployment
- **Code Quality:** SonarQube integration for static code analysis

## Tech Stack

- **Language:** Java
- **Framework:** Spring Boot, Lombok
- **Database:** MySQL
- **Containerization:** Docker
- **CI/CD:** GitLab CI/CD Pipelines
- **Testing:** JUnit (unit & integration), Cypress (E2E), Google Lighthouse (performance)
- **Code Quality:** SonarQube
